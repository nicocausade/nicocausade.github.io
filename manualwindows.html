¿QUE ES UN SISTEMA OPERATIVO?

Un sistema operativo es el software que coordina y dirige todos los servicios y aplicaciones que utiliza el usuario en una computadora.

Requisitos

Para instalar Windows 11, tu pc debe cumplir con ciertos requisitos minimos de hardware

  • PROCESADOR: De 64 bits con 2 o mas nucleos y una frecuencia de 1GHz o mas.
  • MEMORIA RAM: 4GB o mas.
  • ALMACENAMIENTO: 64 GB o mas de espacio disponeble en disco.
  • FIRMWARE DEL SISTEMA: UEFI con arranque seguro.
  • TPM: Modulo de plataforma segura (TPM) version 2.0
  • TAJETA GRAFICA: Compatible con DirectX 12 o posterior con controlador WDDM 2.0
  • CONEXION A INTERNET: Requerida para la configuracion inicial en Windows 11

VERIFICACION DE COMPATIBILIDAD

Si no estas seguro de si tu PC cumple con los requisitos, puedes utilizar la aplicacion"Comprobacion de estado del PC" de Microsoft para verificar la compatibilidad

Metodo 1 - Windows Update

Si tu pc cumple con los requisitos necesarios, puedes actualizar a Windows 11 desde Windows Update como si fuera una actualizaciòn mas

  • Ve a Inicio > Configuración > Windows Update.
  • Haz clic en Buscar actualizaciones.
  • Si la actualización de Windows 11 está disponible, haz clic en Descargar e instalar.
  • Sigue las instrucciones en pantalla para completar la instalación, lo que podría incluir reiniciar el equipo.

    • Metodo 2 - USB Booteable

    Si tu PC no cumple con los requisitos, puedes crear un dispositivo USB booteable de Windows 11 con aplicaciones como Rufus

  • Descarga la imagen ISO desde la página de descarga de Windows 11 de Microsoft (Link aquì).
  • Crea un USB booteable utilizando Rufus para transferir la imagen ISO a una unidad USB (de al menos 8 GB) y hacerla arrancar (Descarga Rufus aquì).
  • Inserta el USB en el equipo y arráncalo desde este medio. Para ello, deberás seleccionar el dispositivo en el menú de arranque.
  • Una vez que el equipo arranque desde el medio, aparecerá la ventana de configuración de Windows 11. Sigue los pasos de la guía e instrucciones en pantalla para completar la instalación.

    • Y listo, ya tendras Windows 11 listo para usar
