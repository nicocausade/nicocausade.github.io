MANUAL PARA INSTALAR EL SISTEMA OPERATIVO LINUX MINT MATE

Hay multiples versiones de Linux Mint, pero en este manual te enseño a descargar Mint Mate

¿QUE ES UN SISTEMA OPERATIVO?

Un sistema operativo es el software que coordina y dirige todos los servicios y aplicaciones que utiliza el usuario en una computadora.

Requisitos

Para instalar Linux Mint Mate, tu pc debe cumplir con ciertos requisitos minimos de hardware

  • Procesador: se requiere al menos un procesador x86_64 (64 bits)
  • Memoria ram: 2 GB de RAM (se recomiendan 4 GB)
  • Espacio: 20 GB de espacio libre en disco (se recomiendan 100 GB)
  • Resolución: Una resolución de pantalla mínima de 1024x768.
  • Ademas se necesita una memoria USB de 16 GB para crear el medio de instalación y una conexión a Internet de banda ancha para descargar la imagen ISO del sistema.

Descargar Linux Mint Mate

  • Descarga la imagen ISO desde la página de Linux Mint (Link aquì).
  • Selecciona MATE
  • Bajà un poco y descargalo desde el primer link
  • Crea un USB booteable utilizando Balena Etcher para transferir la imagen ISO a una unidad USBy hacerla arrancar (Descarga Balena Etcher aquì).
  • Descarga la que mas te guste
  • Presiona "Select Image" y selecciona la imagen de Mint que descargamos previamente
  • Presiona "Select drive" y elige tu memoria USB
  • Inserta el USB en el equipo y arráncalo desde este medio. Para ello, deberás seleccionar el dispositivo en el menú de arranque.
  • Una vez que el equipo arranque desde el medio, se ejecutará Mint desde el usb, para instalarlo deberás ejecutar el programa titulado "Instalar Mint" y seguimos las instrucciones.

    • Y listo, ya tendras Linux Mint listo para usar
